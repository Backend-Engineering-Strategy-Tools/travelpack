# TravelPack

![Minecraft 1.21.4](https://img.shields.io/badge/Minecraft-1.21.4-brightgreen)
![Fabric](https://img.shields.io/badge/Loader-Fabric-blue)
![License: MIT](https://img.shields.io/badge/License-MIT-yellow)

Adds travel utilities for adventurers on the road.

---

## Features

### Sleeping Bag
A lightweight portable bedroll. Use it like a bed to skip the night — no need to place a permanent bed block.

- Stackable up to **16**
- Can be upgraded into a **White Bed**

### Horse Bedroll Slot
Horses have a dedicated **bedroll slot** in their inventory screen alongside the saddle and armour slots. Equip a Sleeping Bag there and it will appear draped across the horse's back.

---

## Crafting

### Sleeping Bag
```
[ Wool ][ Wool ][ Wool ]
```
Any wool colour works.

### White Bed (from Sleeping Bag)
```
[      ][ Sleeping Bag ][      ]
[ Plank][    Plank     ][ Plank]
```

## In-Game Guide

Install [Patchouli](https://modrinth.com/mod/patchouli) and run:

```
/give @p patchouli:guide_book[patchouli:book="travelpack:travelpack_guide"]
```

---

## Installation

1. Install [Fabric Loader](https://fabricmc.net/use/) for Minecraft 1.21.4
2. Install [Fabric API](https://modrinth.com/mod/fabric-api)
3. Drop the TravelPack `.jar` into your `mods/` folder
4. _(Optional)_ Install [Patchouli](https://modrinth.com/mod/patchouli) for the in-game guide book
